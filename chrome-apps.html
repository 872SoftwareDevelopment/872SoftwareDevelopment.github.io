Create a Chrome App

Problem: Too many tabs for different web apps in Chrome are hard to find and organize.

Solution: Create individual icons for webapps that launch in their own chrome containers.

  1. Go to the root or homepage of the webapp.
  2. Click on the vertical ellipsis ( ⋮ ) on the far right of the address bar.
  3. ( ⋮ ) > More Tools > Create Shortcut
  4. Simplify or shorten the title if necessary.
  5. Check the "Open as window" checkbox input.
  6. Click "Create".

The app icon will now appear in your Apps directory, your Launchpad, and app icon Dock. New windows may open in a Chrome tab.
